# Generated default hitsounds

These four mono 16-bit PCM WAV files are original deterministic synthesis made
by `scripts/generate-default-hitsounds.mjs`. They do not contain samples copied
from osu!, McOsu, or any skin. Regenerate them with `npm run generate:hitsounds`.
